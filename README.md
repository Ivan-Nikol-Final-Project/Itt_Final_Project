## Get Started
  * Install Node.js and NPM (should come with)
  * Install MySql
  * Install Composer
  * Instal dependencies(cmd commands)
    * npm install
    * composer update
    * php artisan key:generate
    * php artisan migrate
    * php artisan db:seed
    * php artisan serve
